# Virtual Blood Bank - Frontend

This is a modern React application for the Virtual Blood Bank System.

## Setup Instructions

1.  Open a terminal in this directory:
    ```bash
    cd blood-bank-ui
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at: [http://localhost:5173](http://localhost:5173)

## Features
- **Admin Dashboard**: Overview of donors, requests, and inventory.
- **Secure Authentication**: JWT-based login for administrators.
- **Glassmorphic Design**: Modern, clean, and responsive UI.
